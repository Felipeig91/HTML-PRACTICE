Elementos Semánticos HTML5

¿Qué es HTML5?

HTML5 es la última versión estándar de HTML que introduce nuevos elementos semánticos para mejorar la estructura del contenido.

Ventajas de <details> y <summary>
  • Mejora la accesibilidad
  • Contenido interactivo sin JavaScript
  • Semántica clara
