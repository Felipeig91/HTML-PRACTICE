Input Attributes

Descripción de atributos:
  • autocomplete: sugiere valores guardados por el navegador.
  • autofocus: enfoca el campo al cargar la página.
  • placeholder: muestra un texto de ejemplo dentro del campo.
  • pattern: valida el formato usando expresiones regulares.
  • required: obliga a completar el campo antes de enviar.
  • step: define el incremento para campos numéricos.
  • multiple: permite seleccionar varios archivos.
  • select: permite elegir una opción de una lista desplegable.
  • width / height: ajusta el tamaño de un input tipo imagen.
  • form: conecta un campo a un formulario distinto por su id.
  • formaction / formmethod / formenctype / formtarget / formnovalidate: configuran el envío del formulario desde un botón específico.
