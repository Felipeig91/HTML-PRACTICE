Introducción a SVG

SVG significa Gráficos Vectoriales Escalables. Es un formato basado en XML para crear y mostrar gráficos vectoriales en la web. A diferencia de las imágenes rasterizadas como JPEG o PNG, los gráficos SVG pueden escalar sin perder calidad. Esto hace que SVG sea ideal para logotipos, iconos, ilustraciones y otros gráficos que deben adaptarse a diferentes tamaños de pantalla.

SVG permite la creación de gráficos complejos usando formas como <circle>, <rect>, <line> y <path>. También admite el estilo con CSS y la interactividad con JavaScript, lo que lo convierte en una opción poderosa para el diseño web moderno.
